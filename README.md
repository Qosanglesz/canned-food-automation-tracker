# canned-food-automation-tracker

## How to run the web application
### 1. Change directory to backend-nestjs
```bash
    cd backend-nestjs
 ```

### 2. Install requirements dependencies
```bash
    npm install
```

### 3. Create .env file to ./backend-nestjs
```bash
    touch .env # MAC OS
```
```bash
    PORT = "8000"
```

### 4. Running sever using NPM script
```bash
    npm run start:dev
```
